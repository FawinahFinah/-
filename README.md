# 🎮 课堂游戏库

> 寓教于乐 · 让每一堂课都充满活力

一个由多个 **HTML 课堂游戏** 与 **Excel 题库** 组成的课堂互动工具集，开箱即用、纯前端实现，无需服务器，离线可用。

---

## 📁 项目结构

```
.
├── index.html                  # 主页（游戏库入口）
├── games/
│   ├── quiz/index.html         # 🧠 答题竞赛
│   ├── roll-call/index.html    # 🎡 随机点名
│   ├── matching/index.html     # 🔗 单词配对
│   ├── memory/index.html       # 🃏 记忆翻牌
│   └── timer/index.html        # ⏱️ 课堂计时器
└── question-banks/
    ├── 英语词汇题库.xlsx
    ├── 数学计算题库.xlsx
    └── 常识百科题库.xlsx
```

---

## 🕹️ 游戏说明

| 游戏 | 简介 | 亮点 |
|------|------|------|
| 🧠 答题竞赛 | 单选题限时对战，计分排名 | 支持导入 Excel 题库，越快答越高分 |
| 🎡 随机点名 | 炫酷转盘随机抽取学生 | 自定义名单，点名记录 |
| 🔗 单词配对 | 将词语与解释配对连线 | 支持自定义配对内容，计时挑战 |
| 🃏 记忆翻牌 | 经典翻牌配对游戏 | 支持自定义卡片，4种难度 |
| ⏱️ 课堂计时器 | 倒计时 / 正计时 | 提示音、警示色、记录功能 |

---

## 📚 题库格式

Excel 题库（用于**答题竞赛**游戏导入）须包含以下列：

| 题目 | 选项A | 选项B | 选项C | 选项D | 正确答案 |
|------|-------|-------|-------|-------|---------|
| 问题文本 | 选项1 | 选项2 | 选项3 | 选项4 | A/B/C/D |

---

## 🚀 使用方法

1. **直接打开** `index.html` 文件（双击或用浏览器打开）
2. 点击游戏卡片进入对应游戏
3. 如需自定义题目，下载题库 Excel，按格式修改后在游戏中导入
4. 推荐全屏投屏使用

> 所有游戏均为纯 HTML/CSS/JavaScript 实现，无需网络连接，无需安装任何软件。

## 🎬 全局升级

### 切镜动画升级

**旧效果**：
- 黑色径向透明蒙版
- 白色倾斜带状条纹平移

**新效果** ✨：
- **上下斜向黑色蒙层缓缓展开**
- 左上角斜向蒙层向左上收缩（`clip-path` 动画）
- 右下角斜向蒙层向右下收缩（`clip-path` 动画）
- 中央白色条纹保持不变（字体显示）
- 总耗时：1.2s（对比原1.0s，放缓以显示展开细节）

**实现代码**：
```css
.cutin-mask::before { /* 左上斜向蒙层 */
  animation: tileMaskOpenLeft 1.2s cubic-bezier(0.2, 0.8, 0.2, 1);
}
.cutin-mask::after { /* 右下斜向蒙层 */
  animation: tileMaskOpenRight 1.2s cubic-bezier(0.2, 0.8, 0.2, 1);
}
@keyframes tileMaskOpenLeft {
  0%: clip-path: polygon(0% -50%, 100% 50%, 100% 100%, 0% 100%);
  100%: clip-path: polygon(0% -150%, 100% -50%, 100% 0%, 0% 0%);
}
@keyframes tileMaskOpenRight {
  0%: clip-path: polygon(0% 0%, 100% 0%, 100% 50%, 0% -50%);
  100%: clip-path: polygon(0% 100%, 100% 100%, 100% 150%, 0% 50%);
}
```

## 🧪 测试建议

### 关键验证点

1. **视觉清晰度**
   - [ ] 宝剑像素是否清晰（image-rendering: pixelated）
   - [ ] 山体分层是否清晰
   - [ ] 穿孔和裂纹是否可见

2. **动画流畅度**
   - [ ] 火箭5条是否并行无卡顿
   - [ ] 天雷3颗是否均匀下落
   - [ ] 切镜蒙层是否流畅展开

3. **音视频同步**
   - [ ] 音效与视觉特效启动时刻是否吻合
   - [ ] 多发射火箭的音效是否分离对应
   - [ ] 天雷音效与卡片冲击是否同步

4. **跨浏览器兼容**
   - [ ] Chrome/Edge（clip-path支持）
   - [ ] Firefox（animation-delay支持）
   - [ ] Safari（webkit前缀检查）

5. **性能指标**
   - 帧率：目标 ≥ 45 fps（特效期间）
   - GPU使用：不超过旧版本 20%
   - 内存占用：不超过 5MB 增量

### 降级方案

若浏览器不支持某些CSS特性：

| 特性 | 备选方案 |
|------|---------|
| `clip-path` | 移除蒙层展开效果，保留白条纹 |
| `image-rendering: pixelated` | 使用模糊滤镜代替像素效果 |
| `backdrop-filter` | 移除模糊效果，使用半透明背景 |
| `mix-blend-mode` | 强制 `opacity` 替代 |

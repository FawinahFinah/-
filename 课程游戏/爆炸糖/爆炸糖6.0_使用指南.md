# 爆炸糖6.0 版本 - 分层题库系统使用指南

## 🎯 新增功能概览

爆炸糖6.0版本集成了**智能分层题库系统**，相比之前版本，提供了更智能、更灵活的Excel导入流程。

### 核心改进

| 特性 | 旧版本 | 6.0版本 |
|------|-------|---------|
| **Excel导入** | 固定列映射 | 智能列映射向导 |
| **列配置** | 手动选择 | 自动检测+勾选+拖拽 |
| **题库结构** | 平面列表 | 多级分层树 |
| **用户友好度** | 中等 | 很高（小白友好） |

## 📋 使用步骤

### 第一步：打开游戏
1. 用浏览器打开 `爆炸糖6.0.html` 文件
2. 点击【📦 题库箱】打开侧边栏

### 第二步：导入Excel题库（新流程）

#### 方式A：下载模板并填充
1. 点击【⬇️ 下载模板】获取标准Excel模板
2. 按照模板格式填入题目数据
3. 点击【⬆️ 导入Excel】选择你填充好的Excel文件

#### 方式B：使用现有Excel（必须有标题行）
- Excel格式要求（至少5列）：
  - A列：类型（实词、通假字等）
  - B列：年级（七上、八下等）
  - C列：篇目（《论语》、《庄子》等）
  - D列：原句（包含**高亮**的考核字）
  - E列：注释（对应的释义）
  - F列及以后：可选，用于旧版题库配置

### 第三步：配置列映射（关键步骤）

导入Excel后，会自动弹出**列映射对话框**，这是爆炸糖6.0的核心新功能。

#### 📊 对话框说明

```
┌─────────────────────────────────────────────┐
│ 📋 智能列映射向导                           │
├─────────────────────────────────────────────┤
│                                             │
│ 💡 使用提示：                              │
│ ✅ 自动识别您的Excel列                      │
│ ✅ 勾选☑决定哪些列用于分类                  │
│ ✅ 最后两列已自动设为"题目"和"答案"        │
│ ✅ 可拖拽列名调整分类优先级                  │
│                                             │
│ 📊 您的Excel有以下列：                     │
│ ☑ 📌 类型 (第0列) 【筛选列】              │
│ ☑ 📌 年级 (第1列) 【筛选列】              │
│ ☑ 📌 篇目 (第2列) 【筛选列】              │
│ ☑ 📌 原句 (第3列) 【题目】                │
│ ☑ 📌 注释 (第4列) 【答案】                │
│                                             │
│ ✅ 配置完整：第0列 + 第1列 + 第2列 |       │
│    题目：第3列 | 答案：第4列               │
│                                             │
│ 📝 给此配置起名：[按类型和篇目分类]        │
│                                             │
│ ❌ 取消    ✅ 确认映射                     │
└─────────────────────────────────────────────┘
```

#### 🎮 交互方式

**① 勾选框（决定是否使用该列）**
- 点击左侧☑勾选框，决定该列是否参与分类
- 最后两列会自动勾选为"题目"和"答案"
- 其他列默认勾选为"筛选列"

**② 拖拽排序（调整分类优先级）**
- 如果有多个筛选列（如：类型、年级、篇目）
- 可以**拖拽列名**来调整它们的优先级
- 优先级决定了题库树的层级结构

**③ 实时预览**
- 绿色框显示配置完整✅
- 红色框显示配置不完整⚠️
- 预览会实时更新

**④ 配置命名**
- 可选项，为这个配置起个名字便于后续切换
- 如果不填，系统会自动生成名称

### 第四步：确认映射

1. 配置完成后，点击【✅ 确认映射】
2. 系统会自动生成分层题库树
3. 题库被保存到浏览器本地（localStorage）
4. 弹出成功提示：✅ 配置"xxx"已保存！题库已生成。

### 第五步：开始游戏

1. 题库侧边栏会显示新导入的题库
2. 点击题库名称加载题目
3. 点击【🚀 载入生成】或【⛏️ 开始探险】开始游戏
4. 游戏流程与之前版本相同

## 🎨 列映射UI详解

### 自动默认规则

**爆炸糖6.0的智能检测逻辑：**
```
┌─ 最后一列（倒数第1列）→ 【答案】
│
├─ 倒数第二列         → 【题目】
│
└─ 前面所有列          → 【筛选列】（可调整优先级）
```

**例子1：5列表格**
```
A列: 类型    → 【筛选列1】
B列: 年级    → 【筛选列2】
C列: 篇目    → 【筛选列3】
D列: 原句    → 【题目】（倒数第2列）
E列: 注释    → 【答案】（倒数第1列）
```

**例子2：4列表格**
```
A列: 题目名  → 【筛选列1】
B列: 分类    → 【筛选列2】
C列: 问题    → 【题目】（倒数第2列）
D列: 答案    → 【答案】（倒数第1列）
```

## ❓ 常见问题

### Q1: 为什么拖拽没有反应？
**A:** 确保：
- 你是在**列表项上**拖拽，而不是在复选框上
- 浏览器支持HTML5拖拽（现代浏览器都支持）
- 鼠标按住不松开的情况下拖动

### Q2: 我的Excel格式不符合模板？
**A:** 没问题！爆炸糖6.0支持任意列数的Excel：
- 可以有4列、5列、6列...任意数量
- 只要有最后两列作为题目和答案
- 前面的列都会自动识别为筛选列

### Q3: 能否改变列的顺序？
**A:** 可以！通过**拖拽**：
1. 在列映射对话框中拖拽列名
2. 调整筛选列的优先级
3. 确认后题库树会按新顺序重新生成

### Q4: 多次导入会覆盖吗？
**A:** 不会。每次导入可以：
- 起个不同的配置名称
- 系统会保存为不同的题库配置
- 在侧边栏可以切换使用不同的配置

### Q5: 本地保存在哪里？
**A:** 保存在浏览器的localStorage中：
- 关闭浏览器仍会保留
- 清空浏览器缓存会丢失
- 可以用【📤 导出备份】导出为Excel作为备份

## 🛠️ 故障排查

| 症状 | 原因 | 解决方案 |
|------|------|---------|
| Excel导入后没有弹出对话框 | 浏览器组件加载中 | 等待5秒重试，或刷新页面 |
| 对话框显示不完整 | 浏览器窗口太小 | 最大化浏览器窗口或调整缩放 |
| 拖拽列名没反应 | 浏览器兼容性问题 | 更新浏览器到最新版本 |
| 配置保存失败 | localStorage已满 | 清空浏览器缓存或导出备份后清空 |
| 游戏加载题目失败 | 配置选择错误 | 重新点击题库名称重新加载 |

## 💾 数据备份与恢复

### 导出备份
1. 点击【📤 导出备份】
2. 系统会生成包含所有题库的Excel文件
3. 保存到电脑本地作为备份

### 恢复数据
1. 用【⬆️ 导入Excel】重新导入备份文件
2. 按照列映射流程重新配置
3. 数据恢复完成

## 📚 高级用法

### 场景1：多个课程使用不同的题库结构

**例子：** 七年级和高二使用不同的分类方案

1. 第一次导入：选择【按年级分类】配置
   - 筛选列优先级：年级 > 篇目
2. 第二次导入：选择【按篇目分类】配置
   - 筛选列优先级：篇目 > 年级
3. 在侧边栏可以随时切换使用

### 场景2：导入旧版Excel并升级

如果你有旧版本的Excel（带有题库配置列）：

1. 用爆炸糖6.0导入
2. 在列映射对话框中**取消勾选**旧的题库配置列
3. 按新的分类方案重新组织
4. 导出新格式的Excel作为标准模板

## 💻 技术实现细节

### 新增CSS样式
```css
.col-config-modal                 /* 模态框容器 */
.col-config-header                /* 标题样式 */
.col-preview / .col-preview-item  /* 数据预览 */
.col-mapping-friendly             /* 友好列表容器 */
.col-mapping-item                 /* 单个列项 */
.col-mapping-checkbox             /* 复选框 */
.col-mapping-label                /* 标签 */
.col-mapping-role                 /* 角色标签 */
.qb-config-summary               /* 配置预览框 */
.col-config-actions               /* 按钮组 */
```

### 新增JavaScript函数
1. **showColConfigDialog(jsonData)**
   - 显示列映射向导界面
   - 自动检测和设置默认角色
   - 渲染交互元素

2. **toggleColMapping(col, isChecked)**
   - 处理勾选框变化
   - 更新列角色映射

3. **updateConfigSummary()**
   - 实时显示配置预览
   - 验证配置完整性
   - 更改样式反馈状态

4. **closeColConfigModal()**
   - 关闭对话框
   - 清除待处理数据

5. **confirmColConfig()**
   - 验证配置有效性
   - 调用树生成函数
   - 保存配置到localStorage
   - 显示成功提示

6. **buildAndSaveQBTree(jsonData, qbConfig)**
   - 根据配置生成多层级树
   - 计算每层题目数量
   - 保存完整数据结构

7. **HTML拖拽事件处理**
   - dragstart / dragend
   - dragover / dragleave
   - drop事件处理

### 新增HTML元素
```html
<div id="col-config-modal-overlay">           <!-- 模态框容器 -->
<div id="col-mapping-friendly">               <!-- 列列表 -->
<div id="qb-config-summary">                  <!-- 配置预览 -->
<input id="qb-config-name-input">             <!-- 配置命名 -->
```
